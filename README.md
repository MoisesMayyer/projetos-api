# Estudos de APIs com Python

Este repositório reúne pequenos projetos desenvolvidos durante meus estudos sobre **consumo de APIs utilizando Python**.

O objetivo é aprender, na prática, como aplicações se comunicam com serviços externos, realizando requisições HTTP, manipulando respostas em JSON e integrando diferentes APIs ao código.

##  Tecnologias

- Python 3
- Requests
- JSON
- APIs REST

##  Estrutura do repositório

Cada pasta representa um mini projeto independente, utilizando uma API diferente para praticar conceitos específicos.

Exemplo:

```text
estudos-api/
├── 01-weatherapi/
├── 02-viacep/
├── 03-pokeapi/
├── 04-coingecko/
└── ...
```

##  Conceitos praticados

- Requisições HTTP (`GET`, `POST`, `PUT`, `PATCH` e `DELETE`)
- Consumo de APIs REST
- Utilização de parâmetros em requisições
- Autenticação com API Key
- Manipulação de respostas JSON
- Tratamento de erros e códigos de status HTTP
- Organização e modularização de código em Python

##  Objetivo

Este repositório documenta minha evolução no estudo de APIs por meio de diversos projetos pequenos e práticos.

Conforme avanço nos estudos, novos projetos serão adicionados utilizando diferentes serviços e documentações de APIs.

##  Segurança

Algumas APIs exigem uma **API Key** para autenticação.

Por segurança, chaves de acesso não são enviadas para o GitHub. Elas devem ser configuradas localmente utilizando variáveis de ambiente ou arquivos `.env`.
