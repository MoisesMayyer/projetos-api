# Pokédex CLI

Uma aplicação de terminal desenvolvida em **Python** que consome a **PokéAPI** para consultar informações de qualquer Pokémon.

Este projeto faz parte da minha coleção de estudos sobre consumo de APIs REST utilizando Python.

---

## Tecnologias

- Python 3
- Requests
- Rich
- JSON
- Git & GitHub

---

## Funcionalidades

-  Buscar Pokémon pelo nome ou ID
-  Exibir informações detalhadas
-  Salvar histórico de consultas
-  Visualizar histórico
-  Interface moderna utilizando Rich
-  Tratamento de erros

---

##  Informações exibidas

- Nome
- Número da Pokédex
- Tipo(s)
- Altura
- Peso
- Experiência Base
- Habilidades
