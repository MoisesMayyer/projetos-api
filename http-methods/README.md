# HTTP Methods Practice

Exercícios de prática dos principais métodos HTTP utilizando Python e a biblioteca Requests.

O objetivo deste módulo é entender como funciona a comunicação entre um cliente e uma API REST, praticando operações de criação, consulta, atualização e remoção de dados.

## Métodos estudados

### GET
Utilizado para buscar informações de uma API.

Exemplo:
- Listar usuários
- Buscar um registro específico

### POST
Utilizado para enviar novos dados para uma API.

Exemplo:
- Criar um usuário
- Criar um registro

### PUT
Utilizado para atualizar completamente um recurso existente.

Exemplo:
- Substituir todos os dados de um usuário

### PATCH
Utilizado para atualizar parcialmente um recurso.

Exemplo:
- Alterar apenas um campo de um registro

### DELETE
Utilizado para remover um recurso.

Exemplo:
- Excluir um usuário ou registro

## Tecnologias utilizadas

- Python
- Requests
- JSON
- APIs REST

## Objetivo

Praticar o funcionamento das requisições HTTP e entender a base necessária para criação e consumo de APIs.
