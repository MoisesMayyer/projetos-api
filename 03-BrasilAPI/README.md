# BrasilAPI - Consulta de CEP

Projeto desenvolvido em Python utilizando a BrasilAPI para consulta de endereços através do CEP.

O objetivo do projeto é praticar consumo de APIs, tratamento de dados JSON, organização de código e persistência de informações utilizando arquivos JSON.

## Funcionalidades

- Consulta de endereço por CEP
- Integração com a BrasilAPI
- Exibição dos dados do endereço
- Salvamento do histórico de consultas
- Leitura do histórico salvo

## Tecnologias utilizadas

- Python
- Requests
- JSON
- BrasilAPI
