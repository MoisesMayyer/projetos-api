🌤 Weather API - Consulta de Clima

Projeto desenvolvido em Python para praticar o consumo de uma **API REST** utilizando a WeatherAPI.

A aplicação realiza requisições para uma API externa, recebe os dados em formato JSON e exibe informações climáticas de uma cidade escolhida pelo usuário.

##  Tecnologias utilizadas

- Python 3
- Requests
- Python-dotenv
- JSON
- WeatherAPI

##  Funcionalidades

-  Consultar clima atual de uma cidade
-  Exibir informações meteorológicas
-  Alterar cidade de consulta
-  Salvar histórico de consultas
-  Visualizar histórico de pesquisas

Estrutura do projeto

```text
01-weatherapi/

├── main.py              # Menu principal do sistema
├── api.py               # Comunicação com a WeatherAPI
├── historico.json       # Dados das consultas realizadas
├── requirements.txt     # Dependências do projeto
└── README.md
```

## Conceitos praticados

- Requisições HTTP com `requests`
- Consumo de APIs REST
- Manipulação de JSON
- Uso de API Key
- Variáveis de ambiente
- Organização de código em módulos
- Persistência de dados utilizando arquivos JSON
- Tratamento de erros
